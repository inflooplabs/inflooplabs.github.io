## Project Overview

`inflooplabs.github.io` is a marketing and support website for a group called Infinite Loop Labs. ILL creates apps for iOS and the Apple ecosystem of devices.

The core tabs of the website include:

* **Home:** The main marketing page for Infinite Loop Labs and the entry point of the website.
* **Criterion:** The main marketing page for Criterion, one of the apps made by ILL.
* **Support:** Contains a few help pages with information about how to resolve issues in ILL's apps, an AI chatbot that reads the pages and tries to help generate solutions to issues, and a form to put in a support ticket with the ILL support team.
* **AI-Powered Assistance:** The website integrates with Gemini 3 to provide support assistance.

## Building and Running

To build and run the project:

1. Open the `index.html` file in a browser, preferably Safari.

## Development Conventions

* **Language:** The project is written in HTML, CSS, and JS.
* **UI:** The user interface is built with HTML.
* **Data Persistence:** No data is stored anywhere.
* **Coding Style:** The code follows standard HTML conventions, including `PascalCase` for types and `camelCase` for functions and variables. In addition, indents are 8 spaces, there are no comments, there are no double newlines (except between top-level objects in CSS/JS), it should be written `}\nelse {` instead of `} else {`, and all braces should be expanded - meaning no `{ number == 0 }`, it should be `{\n        number == 0\n}`. All top-level objects must be in alphabetical order within the file.

## Key Files

* `index.html`: The entry point of the website and ILL's marketing page.
* `criterion.html`: Criterion's marketing page.
* `support.html`: A page containing various features for assistance.
* `privacy-policy.html`: The privacy policy for all of ILL's apps.
* `newsroom.html`: All news and updates provided by ILL - will also be sent to the mailing list.
* `404.html`: The 404 Not Found error page.
* `style.css`: Contains all the styling information for the website.
* `script.js`: Contains all the scripting information for the website.
